# ⚡ FinSight AI: Premium Document Intelligence

FinSight AI is a modern, high-end financial document analysis platform designed to help users identify hidden risks, summarize complex clauses, and provide actionable financial insights from PDFs instantly.

![FinSight Banner](https://img.shields.io/badge/Financial-AI_Copilot-indigo?style=for-the-badge&logoColor=white) 
![Stack](https://img.shields.io/badge/React_|_FastAPI_|_Tailwind-purple?style=for-the-badge)

---

## ✨ Key Features

- **🚀 Interactive Risk Slider**: A professional, visual risk assessment of your document mapped to a precise scale (Safe, Medium, High).
- **📋 Actionable Suggestions**: Smart, toggleable checklist of steps to take after document analysis, featuring a premium violet-gradient theme.
- **💬 AI Copilot Chat**: A dedicated full-page chatbot built with an enterprise SaaS layout to answer specific questions about your document's clauses, fees, and fine print.
- **🛡️ Data Privacy**: Secure, 256-bit encrypted transfers. No data is stored, ensuring user privacy and confidentiality.
- **🌐 Advanced AI Analysis**: Powered by Groq API (Llama 3.3 70B) for ultra-fast and intelligent reasoning.

## 🛠️ Technology Stack

### Frontend
- **React 18** + **Vite** (High Performance)

- **React Icons** (Professional Visuals)

### Backend
- **FastAPI** (Python High-Speed Framework)
- **PyPDF2** (Document Processing)
- **OpenAI / Groq API** (Advanced Financial Reasoning Models)

## 🏃 Getting Started

### 1. Prerequisites
- Python 3.9+ 
- Node.js 18+
- Groq/OpenAI API Key

### 2. Backend Setup
```bash
# Navigate to backend 
cd backend

# Create virtual environment 
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate

# Install dependencies 
pip install -r requirements.txt

# Run the server 
python -m uvicorn main:app --reload
```

### 3. Frontend Setup
```bash
# Navigate to frontend 
cd frontend

# Install dependencies 
npm install

# Run the development server 
npm run dev
```
